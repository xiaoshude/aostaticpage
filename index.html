W同学

录取院校：澳洲国立大学

留学顾问：白杨

案例难点 :

学历低、成绩不理想选定的学校和专业要求高

学生感言 :

我深知我的情况申请好的学校和专业难度极大，但是白老师凭借自己的经验能力，让我获得面试机会并拿下Offer，我很幸运遇到白老师帮我申请成功。

M同学

录取院校：阿德雷德大学

留学顾问：饶勤

案例难点 :

无学位证和毕业证、定位申请生物专业、大学有挂科

学生感言 :

留学本来没报任何希望，当饶老师跟我说阿德雷德给我发了录取还减免了2年学分，我简直不敢相信。饶老师改变了我的人生。

李同学

录取院校：新南威尔士大学

留学顾问：赵于兰

案例难点 :

gpa2.9、无雅思成绩、申请时无法提供毕业证

学生感言 :

在我各种条件不是很理想，又不是很配合、中途又变国家的情况下，赵老师一直耐心服务，帮我申请到新南威，我非常满意。

H同学

录取院校：墨尔本大学

留学顾问：成戎

案例难点 :

有移民想法、非211院校、申请专业受限

学生感言 :

成老师真的是非常专业，为我制定的计划非常明确。在选择老师，老师的专业度和经验真的是非常重要。

王同学

录取院校：莫那什大学

留学顾问：卜振华

案例难点 :

有被拒经历、时间紧迫、无法提供相关证明

学生感言 :

我把一切希望都寄托在了卜老师身上，卜老师真的是非常给力，快速帮我制定好计划，很快就完了申请拿到了录取。

Y同学

录取院校：悉尼大学

留学顾问：李宝红

案例难点 :

非211大学毕业、无雅思、不清楚留学目的和将来规划

学生感言 :

对留学没有一点概念，李老师耐心的给我讲解和指导，并从我未来的发展方面帮我规划留学，感谢李老师对我的帮助。
